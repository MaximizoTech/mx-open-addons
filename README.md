# MaximizoTech Open Addons (mx-open-addons)

Welcome to the official public repository for free and open-source Odoo modules developed by **MaximizoTech**. 

Our goal is to contribute back to the fantastic Odoo community by providing clean, professional, and highly useful modules for Odoo 18.0 and beyond.

##  Available Modules

| Module | Version | Description |
|--------|---------|-------------|
| [Odoo Branding & Menu Manager (`mx_odoo_interface`)](./mx_odoo_interface) | 18.0 | A comprehensive interface tool to remove Odoo branding, clean up user menus, and control menu visibility on a per-user basis. |
| [POS Rebranding (`mx_pos_rebranding`)](./mx_pos_rebranding) | 18.0 | All-in-one POS customization: branding, theme colors, refund security, orderline management, and custom receipts. |

##  Installation

1. Clone this repository into your Odoo instance's `addons` path or `custom_addons` directory:
   ```bash
   git clone https://github.com/your-username/mx-open-addons.git
   ```
2. Restart your Odoo server.
3. Activate **Developer Mode** in your Odoo instance.
4. Go to **Apps** -> **Update Apps List**.
5. Search for the module name you wish to install and click **Install**.
